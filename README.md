# LLWS Escalation Monitoring Prototype

A Streamlit dashboard prototype for monitoring probabilistic Low-Level Wind Shear (LLWS) escalation risk using multi-runway AWOS surface observations.

This dashboard is a portfolio-oriented decision-support prototype and is not an operational certified warning system.

## Features

- Probabilistic LLWS escalation monitoring
- Raw and smoothed escalation probability timeline
- Operational alert status based on threshold, smoothing, persistence, and hold logic
- Demo window visualization
- Confusion matrix and final checkpoint performance summary

## Notes

This repository uses a small public demo dataset extracted from the final project checkpoint. The original full project data and model development files are not included.
